# Распознанный текст

Здесь хранится человекочитаемая расшифровка информации с фотографий.

- `visual-transcription.md` - ручная визуальная расшифровка видимых записей.
- Техническая JSON-версия находится в `../../technical-data/extracted/visual-transcription.json`.
- Сырые Tesseract-файлы, если нужны для сравнения, находятся в `../../technical-data/ocr/raw-text/`, но они не являются основой финальной расшифровки.

Сомнительные рукописные места помечены как неуверенные и должны сверяться с оригиналами.
