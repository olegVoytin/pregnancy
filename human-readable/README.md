# Материалы беременности

Эта папка содержит читаемую версию предоставленных материалов.

- `index.html` открывает локальную галерею в браузере.
- `images/` содержит JPEG-версии исходных HEIC-файлов.
- Оригиналы без изменений находятся в `../technical-data/source-images/`.

На этом шаге данные только организованы и визуально подготовлены. Распознавание текста и медицинская интерпретация не выполнялись.
